GitHub SSH + Repo Workflow

Minimal command reference for running two GitHub accounts with verified commits.

1. Primary Account (Global Setup)

git config --global user.name "YourPrimaryUser"
git config --global user.email "YOUR_GITHUB_ID+YourPrimaryUser@users.noreply.github.com"
git config --global gpg.format ssh
git config --global commit.gpgsign true
git config --global tag.gpgsign true
git config --global user.signingkey ~/.ssh/id_ed25519.pub

2. Secondary Account (Per Repository)

git config user.name "YourSecondUser"
git config user.email "YOUR_GITHUB_ID+YourSecondUser@users.noreply.github.com"
git config user.signingkey ~/.ssh/id_ed25519_second.pub
git config gpg.format ssh
git config commit.gpgsign true

3. SSH Remote

git remote add origin git@github.com:YourUser/your-repository.git

4. GitHub CLI Authentication

gh auth login

Choose SSH when prompted.

5. Repo Overrides Global Config

.git/config

6. Debug

git config --list --show-origin
git log -1 --show-signature
ssh -T git@github.com

1. Create Project Folder

mkdir my-project
cd my-project

2. Init Project

git init
git add .
git commit -m "initial commit"

3. Create Remote

gh repo create my-project \
--source=. \
--remote=origin \
--private \
--push

Template Repo

gh repo create my-project \
--template owner/template-repo \
--private

One-Line Full Init

mkdir my-project && cd my-project && git init && git add . && git commit -m "initial commit" && gh repo create my-project --source=. --remote=origin --private --push

Link Existing Remote

git remote add origin git@github.com:YourUser/my-project.git
git branch -M main
git push -u origin main

Daily

git add .
git commit -m "update"
git push
